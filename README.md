Oracle JD Edwards EnterpriseOne Log reader

Functionality:
- read all JDE E1 logs
- parse into excel format

Usage:
- Build directory structure as follows:
    * C:\JDE_E1_logs\
    * C:\JDE_E1_logs\xlsx
    * C:\JDE_E1_logs\logs
- Put your E1 logs into logs subfolder
- Run
